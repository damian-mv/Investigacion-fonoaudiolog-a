# investigacion-fonoaudiologia

Primera versión Quarto Book para estudiantes de fonoaudiologia.

## Archivos principales

- `_quarto.yml`: configuracion general del libro.
- `index.qmd`: prefacio y orientacion de uso.
- `01-07-*.qmd` y `04-revisiones.qmd`: capitulos tematicos del libro.
- `references.bib`: bibliografia centralizada.
- `apendice-r.qmd`: guia breve de reproducibilidad en R.

Link libro: https://damian-mv.github.io/Investigacion-fonoaudiolog-a/
