# investigacion-fonoaudiologia

Base inicial de un Quarto Book para estudiantes de fonoaudiologia.

## Archivos principales

- `_quarto.yml`: configuracion general del libro.
- `index.qmd`: prefacio y orientacion de uso.
- `01-07-*.qmd`: capitulos tematicos.
- `references.bib`: bibliografia centralizada.
- `apendice-r.qmd`: guia breve de reproducibilidad en R.

## Notas

- `apa.csl` se dejo como marcador tecnico local para no romper la estructura del libro; conviene reemplazarlo por el estilo APA oficial antes de la version final.
- Algunas referencias pedidas por el prompt quedaron marcadas como `[VERIFICAR]` para evitar inventar datos bibliograficos.
